pressureNET Developers

We offer a pressureNET API and SDK to developers. Our Live API allows access to our global dataset of atmospheric pressure with simple HTTPS requests. Our SDK is open source on GitHub and enables developers to include pressureNET’s core functionality inside their apps to collect and distribute Android sensor data.

We’re going to ensure that our live data is as accessible as possible. If there’s anything that we can do to make the API service or SDK package easier to use, please let us know as we intend to improve them based on your feedback.

Documentation is available for the API (pdf) as well as the SDK. In order to use these services, please sign up for an API key below.
